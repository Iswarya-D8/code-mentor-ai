Code Mentor AI

An AI-powered web application that helps analyze, explain, and optimize code.

🚀 Features

- Code explanation
- Error detection and fixing
- Complexity analysis
- Code optimization

🛠️ Tech Stack

- Frontend: Vite + React (TypeScript)
- Styling: Tailwind CSS
- Backend: Supabase
- AI: Transformers / ML models

▶️ How to Run

npm install
npm run dev

📁 Project Structure

- src/ → main application code
- public/ → static files
- supabase/ → backend config

📌 Notes

Make sure to create a ".env" file for API keys before running.
